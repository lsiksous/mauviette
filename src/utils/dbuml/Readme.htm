DB-UML Modeling Tool Release 1.0.4

DB-UML is an open source modeling tool for creating, updating and visualizing relational database schemas.  Using the tool you can create a new model of database elements, import relational catalog meta-data into a model and generate SQL create statements from a model.

This release is compatible with ArgoUML release 2.4. 

 

For installation instructions see the User Guide.  To see how UML is used to model database elements see the UML Database Modeling document.
